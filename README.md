# Hidder

Hidder là bản mobile app local-first theo hướng stealth vault.

## Features

- Setup flow không cần login/signup
- Notes disguise shell
- Real code unlock vào private vault
- Decoy code unlock vào decoy vault
- Quick Escape quay về shell public
- Private notes local
- Decoy notes local
- Photo vault import local
- Auto-lock khi app vào background

## Run

```bash
npm install
npx expo start
```

## Tech

- Expo
- React Native
- AsyncStorage
- SecureStore
- LocalAuthentication
- ImagePicker
- FileSystem
